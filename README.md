# tool-kit

> tool-kit for programer

## Feature

* base64
* UrlEncode/UrlDecode
* JSON/xml
* qrcode
* MD5
* jpg/png compress
* ip dns
* UglifyJs
* unicode


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
